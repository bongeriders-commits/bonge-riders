Seed the feed
Pull free stock clips from Pexels and post them as veeds — for populating the feed before real riders start posting.
Only use this for footage you're actually licensed to post. Pexels' license covers commercial use with no attribution required — good for generic filler (roads, riders, city life, nature). It won't have "funny viral" content; that's almost always copyrighted elsewhere and shouldn't go through this tool.

This page isn't linked from the app's nav — keep the URL to yourself, and consider deleting it (or password-gating it) before wider rollout.

Background music: Pexels clips are silent, so if a Jamendo key is set above, posting will send each clip to /api/merge-music first, which merges in a mood-matched track before the veed goes live. Merged clips get uploaded to your own Firebase Storage (not linked from Pexels' CDN anymore) — a bit more storage use per seeded clip. Leave the Jamendo key blank or untick the box to skip this and post silent, like before.
